Installing ConMan

  • Mac OS (very experimental)
  • Linux

Linux

First you need to get the client and configure your system. We may at some point provide a script to make this easier but for now:

  • Download the latest release on ConMan and place it in your $PATH.
  • Download the companion conman.desktop file and place it in ~/.local/share/applications/
  • Add a url scheme handler for conman:
    $ xdg-mime default conman.desktop x-scheme-handler/conman

You're all set! Now you can install apps from our app list.
